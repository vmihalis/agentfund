Create `src/agents/governance-agent.ts`:
- Extends BaseAgent with role 'governance'
- Constructor takes AgentEventBus plus IScoutAgent, IAnalyzerAgent, ITreasuryAgent, and an optional Anthropic client (for dependency injection / testing)
- `initialize()`: emits status 'initialized', validates ANTHROPIC_API_KEY exists (warn if missing, don't throw -- fallback will handle)
- `shutdown()`: emits status 'shutdown'
- `executeFundingPipeline(request: { query: string, budget: number }): Promise<DecisionSummary>`:
  1. Emit pipeline:step discover started
  2. Call scout.discoverProposals(request.query)
  3. Emit pipeline:step discover completed with count
  4. For each proposal: emit evaluate started, call analyzer.evaluateProposal, emit evaluate completed with score
  5. Call this.makeDecision(evaluations, request.budget)
  6. Emit pipeline:decision with the decision
  7. For each allocation where action === 'fund': call treasury.executeFunding, emit pipeline:funded
  8. Return the DecisionSummary

- `makeDecision(evaluations: Evaluation[], budget: number): Promise<DecisionSummary>`:
  1. Build a prompt with evaluations and budget context
  2. Define FundingDecisionSchema using Zod (matching RESEARCH.md Pattern 5)
  3. Convert to JSON Schema via zodToJsonSchema
  4. Call client.messages.create with model from ANTHROPIC_MODEL env (default claude-sonnet-4-20250514), tool_use with tool_choice forced to 'submit_decision'
  5. Extract tool_use block, validate with Zod schema
  6. Map to DecisionSummary type
  7. Wrap in try/catch: on ANY error (API, parsing, validation), call this.makeFallbackDecision(evaluations, budget)

- `private makeFallbackDecision(evaluations: Evaluation[], budget: number): DecisionSummary`:
  1. Sort evaluations by overallScore descending
  2. Fund proposals with overallScore >= 7 until budget runs out
  3. Each allocation gets reasoning: "Auto-approved: score {X}/10 exceeds threshold (Claude API unavailable)"
  4. Rejected proposals get reasoning: "Auto-rejected: score {X}/10 below threshold (Claude API unavailable)"
  5. Summary: "Automated decision (Claude API unavailable): funded {N} of {M} proposals based on score threshold"
