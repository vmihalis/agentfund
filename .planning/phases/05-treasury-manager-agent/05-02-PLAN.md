       export interface DlmmPosition {
         positionAddress: string;
         poolAddress: string;
         binIds: number[];

       }

       export interface DlmmPoolInfo {
         poolAddress: string;

    3. Create `src/lib/meteora/dlmm-client.ts`:
       - Class `DlmmClient` with constructor taking `(connection: Connection, cluster: 'devnet' | 'mainnet-beta')`
       - `async createPool(tokenX: PublicKey, tokenY: PublicKey, creator: Keypair): Promise<DlmmResult<{ poolAddress: string }>>`:
         Uses `DLMM.createCustomizablePermissionlessLbPair()` with binStep=10, activeId=8388608 (center), feeBps=25, ActivationType.Timestamp, immediate activation. Signs and sends transaction. Discovers pool address via `DLMM.getCustomizablePermissionlessLbPairIfExists()`. Saves pool address to `keys/dlmm-pool.json`. Returns pool address + signature.
       - `async addLiquidity(poolAddress: PublicKey, user: Keypair, xAmount: number, yAmount: number): Promise<DlmmResult<{ positionAddress: string }>>`:
         Loads pool via `DLMM.create(connection, poolAddress, { cluster })`. Gets active bin. Creates position keypair. Uses `initializePositionAndAddLiquidityByStrategy` with StrategyType.Spot, 10 bins each side, slippage 1. Saves position keypair to `keys/dlmm-positions/{positionPubkey}.json`. Returns position address + signature.
       - `async removeLiquidity(poolAddress: PublicKey, positionPubkey: PublicKey, user: Keypair): Promise<DlmmResult>`:
         Loads pool. Gets position info. Calls `removeLiquidity` with `bps: new BN(10000)` (100%) and `shouldClaimAndClose: true`. Sends all returned transactions sequentially. Returns signatures array.
       - `async getPositions(poolAddress: PublicKey, userPubkey: PublicKey): Promise<DlmmPosition[]>`:
         Loads pool. Calls `getPositionsByUserAndLbPair(userPubkey)`. Maps to DlmmPosition[].
       - `async getPoolInfo(poolAddress: PublicKey): Promise<DlmmPoolInfo | null>`:
         Loads pool. Gets active bin. Returns pool info or null if pool not found.
       - All methods wrap in try/catch and return `{ success: false, error }` on failure. NEVER throw.
       - Use `BN` from `bn.js` for amounts (Meteora SDK requires BN type, not BigInt).
       - Use 10 bins each side (20 total) to stay within Solana's 1232-byte tx limit.
